Shortest Paths

Shortest paths in graphs are a very important problem in many areas of computer science and other disciplines as well. The Visualisation-Tool provided resembles a graph in such a way that every square is a node inside the graph, which is connected to all sourrounding squares (nodes).

How does it work?

1. Select your algorithm; 2. Draw your obstacles; 3. Start the algorithm and see how it finds the shortest path from your set starting point to your set finishing point. Stop the algorithm to examine its current state, maybe draw some more obstacles and then resume execution.
Grey fields are fields the algorithm knows about, but didn't yet fully explore. Orange fields are the ones the algrithm already looked at.
