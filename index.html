Inspiration and future

The idea to write a tool that helps to visualize algorithms first came across my mind when i started to learn about the algorithms themselfes in my second semester of uni. The two types of algorithms i find most interesting in this context are sorting algorithms and algorithms that work on graphs, in particular shortest path algroithms.
The following gif, showing a set of bars of different heights being sorted using heapsort is what finally led me to persue this excersize.

In the future i want to continue on this project and implement more algorithms and their visualization. I find it very easy to understand algorithms and how they work, if they are properly visualized. Playing around with them and trying different things also helps to get a grasp on how they behave.
